Botón por defecto

Clase

Para aplicar el diseño al botón solo colocar la clase b-btn y el atributo color con el color que desees:



Color

Aquí se muestran los botones con los colores disponbles incluyendo algunos colores extras:

Extras:
Para enlaces
Para fondos oscuros
Para fondos claros


bordered

Añade bordes a los botones pero quita el efecto de enfoque.

<button class="b-btn" color="blue" bordered>Botón</button>


no-efx

Quita el efecto de sombreado al pasar sobre el boton.

<button class="b-btn" color="blue" no-efx>Botón</button>


no-focus

Quita el efecto de enfocado en los bordes.

<button class="b-btn" color="blue" no-focus>Botón</button>
