Botones | b-btn-push

Clase

Para aplicar el diseño al botón solo colocar la clase b-btn-push y el atributo color con el color que desees:

Para saber los colores disponibles abre aquí

Atributos

color

Aquí se muestran los botones con los colores disponibles incluyendo algunos colores extras:

<button class="b-btn-push" color="blue">Botón</button>
Para fondos oscuros
Para fondos claros
bordered
Esta propiedad quita el efecto de enfoque.
<button class="b-btn-push" color="blue" bordered>Botón</button>
no-efx
Esta propiedad quita el efecto de sombreado al pasar sobre el botón.
<button class="b-btn-push" color="blue" no-efx>Botón</button>
