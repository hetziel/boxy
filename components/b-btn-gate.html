Botones | b-btn-gate

Clase

Para aplicar el diseño al botón solo colocar la clase b-btn-gate y el atributo color con el color que desees:

Para saber los colores disponibles abre aquí

Atributos

color

Aquí se muestran los botones con los colores disponibles incluyendo algunos colores extras:

<button class="b-btn-gate" color="blue">Botón</button>
no-efx
Esta propiedad quita el efecto de cambiar el color del fondo al pasar sobre el botón.
<button class="b-btn-gate" color="blue" no-efx>Botón</button>
closed
Esta propiedad invierte el efecto de compuerta al pasar sobre el botón.
<button class="b-btn-gate" color="blue" closed>Botón</button>
dir
Esta propiedad se utiliza para configurar la dirección del efecto compuerta

Todavía no está implementado en la propiedad closed

<button class="b-btn-gate" color="blue" dir="1">Botón</button>
