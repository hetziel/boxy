Botones

Boxy actualmente contiene 5 tipos de botones con multiples atributos para personalizarlos a tu gusto:

¿Como usar?

Simplemente añadiendo la clases disponibles con el prefijo b-btn puedes cambiar el aspecto del botón

Escoge un color deseado con el atributo color disponible para los botones.

Para saber los colores disponibles abre aquí

Clases Atributos
  • b-btn (Por defecto)
  • b-btn-push (Presión)
  • b-btn-gate (Compuerta)
  • b-btn-gradient (Gradiente)
  • b-btn-neon (Neón)
  • color
<button class="b-btn" color="blue">Botón</button>
Tipos de botones

Cada botón tiene una clase y atributos propios, dale clic a algún botón para ver como personalizarlos
