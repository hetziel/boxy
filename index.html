Bienvenidos a Boxy

¿Qué es Boxy?

Boxy es una biblioteca multiplataforma que contiene un conjunto de herramientas y un marco de componentes para diseñar sitios y aplicaciones web. Su objetivo es proporcionar todas las herramientas necesarias para crear hermosas aplicaciones web.
