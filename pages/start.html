Comienza con Boxy

Boxy es un Framework orientado al Front-end para tener una apariencia elegante, con multiples funciones para facilitar el desarrollo.

Se ha formulado una estandarización para los tamaños del texto de encabezado.

Inicio rápido

Comience incluyendo el CSS y JavaScript listos para producción de BOXY a través de CDN sin necesidad de ningún paso de compilación

Coloca el trozo de código en el <head> y empieza a usar los nuevos complementos de Boxy en tu proyecto.

<link rel="stylesheet" href="https://boxy.pages.dev/css/boxy.min.css"/>

Coloca el trozo de código antes de terminar el en el </body> para hacer funcionar las modales y bg-img

<script type="text/javascript" src="https://boxy.pages.dev/js/boxy.min.js"></script>
Ejemplo

Crea un archivo HTML y copia este ejemplo para hacer una prueba del CDN de Boxy.

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Bienvenido a Boxy</title>
  <link rel="stylesheet" href="https://boxy.pages.dev/css/boxy.min.css"/>
</head>
<body>
  <h1>Hola Mundo!</h1>
  <script type="text/javascript" src="https://boxy.pages.dev/js/boxy.min.js"></script>
 </body>
</html>
