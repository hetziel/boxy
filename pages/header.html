Header

Para uso de este componente se utiliza la clase: b-header:

Vista Previa Código
<header class="b-header">
</header>

Puedes añadir el nombre o logotipo del sitio con b-header-logo dentro del b-header

Vista Previa Código
BOXY
<header class="b-header">
  <a href="/" class="b-header-logo">BOXY</a>
</header>

Añade botones o links de navegación con b-header-content

Vista Previa Código
BOXY
<header class="b-header">
  <a href="/" class="b-header-logo">BOXY</a>
  <div class="b-header-content">
   <button class="b-btn" color="default">About</button>
   <button class="b-btn" color="default">Portfolio</button>
   <button class="b-btn" color="success">Contact</button>
  </div>
</header>

Añade botones o links de navegación fijos al final con b-header-fixed

Vista Previa Código
BOXY
<header class="b-header">
  <a href="/" class="b-header-logo">BOXY</a>
  <div class="b-header-content">
   <button class="b-btn" color="default">About</button>
   <button class="b-btn" color="default">Portfolio</button>
   <button class="b-btn" color="success">Contact</button>
  </div>
  <div class="b-header-fixed">
   <button class="b-btn" color="default">Login</button>
   <button class="b-btn" color="default">Sign up</button>
   <button class="b-btn" color="default"><i class="fa-brands fa-github"></i> Github</button>
  </div>
</header>

Añade color de fondo con la propiedad global bg

Puede consultar los colores disponibles desde aquí

Vista Previa Código
BOXY
<header class="b-header" bg="blue">

  <div class="b-header-content">
   <button class="b-btn" color="default">About</button>
   <button class="b-btn" color="default">Portfolio</button>
   <button class="b-btn" color="success">Contact</button>
  </div>
</header>

Este complemento tiene una propiedad propia para tener el efecto borroso con skin="blur" tomando en cuenta que el fondo debe de tener una imagen
