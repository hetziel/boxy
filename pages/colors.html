Colores

Boxy tiene un sistema de colores integrado, de manera que todos los componentes tengan la facilidad de colorearse.

Atributos globales

Para dar color a elementos se utiliza el atributo bg="", para textos se utiliza tx=""

Atributos propios

Para dar color a componentes que tienen disponible el atributo color, se utiliza el atributo color="".

  • BG (Fondo)
  • TX (Texto)
  • COLOR (Componente)
