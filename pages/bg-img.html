BG-IMG

Cambia el color de fondo de cualquier contenedor con el atributo bg-img=""

Atributos propios

Utiliza el atributo fixed para tener un fondo fijo aun cuando se haga scroll

Utiliza el atributo contain para que toda la imagen se muestre en el contenedor

Esta función especial requiere Boxy.js

Ejemplos de uso
bg-img="test.jpg"
fixed
contain
