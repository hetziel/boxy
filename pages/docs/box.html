Box

El box es un contenedor de información parecido a una Card. Se llama con la clase b-box

<div class="b-box">
  <div class="b-box-header">Titulo</div>
  <div class="b-box-content">Contenido</div>
</div>
Se usa b-box para el contendedor.
Se usa b-box-header para la cabecera.
Se usa b-box-content para el contenido.
También se puede cambiar el color de fondo con la propiedad bg (las propiedades bg pronto se añadirán a la documentación.)
