Botones

Para incorporar el diseño al elemento del botón solo colocar las clases disponibles para este:

Clase Propiedades propias Valores extras
  • color
  • bordered
  • no-efx
  • no-focus
Propiedad color
<button class="b-btn" color="blue">Botón</button>
Para fondos oscuros
Para fondos claros
Propiedad bordered Esta propiedad quita el efecto de enfoque.
<button class="b-btn" color="blue" bordered>Botón</button>
Propiedad no-efx Esta propiedad quita el efecto de sombreado al pasar sobre el boton.
<button class="b-btn" color="blue" no-efx>Botón</button>
Propiedad no-focus Esta propiedad quita el efecto de enfocado.
<button class="b-btn" color="blue" no-focus>Botón</button>
  • color
  • bordered (Agrega bordes y quita enfoque)
  • no-efx (Quita los efectos hover)
